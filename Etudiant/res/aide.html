Aide d'utilisateur

Emploie Enastic

Emploi Enastic est une simple Applications de visualisations d’emploi du temps de votre classe. Vous pouvez l'utilisez pour consulter l'emploi de temps de votre classe ou d'une autre classe. Son autre fonctionnalité d'impression d’emplois vous permet d'imprimer votre emploi de temps parmis vos fichiers locaux pour les consulter en dehors de l’ applications.

Bouton parametré

Le bouton paramètre permet de configurer le programme pour qu'il se connecte automatiquement à chaque demarrage au serveur d'emploi.
Une fois clicquer dessus, une fenêtre de paramétrage s'affiche avec cinq(5) champs.

  • Le premier champ concerne l'adresse du seveur. En temps normal vous n'aurez pas besoin de le modifier. Mais s'il arrive que l'adresse du serveur soit changé, demander à l'administration la bonne adresse.
  • Le second champ concerne le numéro de port d'écoute du serveur d'emplois. Ne le modifiez pas a moins que l'administration ne vous demande de le faire. Par défaut il est réglé sur le port 2222.
  • Le troisième champ concerne votre filière. Cela permettra au programme de vous afficher automatiquement l'emploi de temps de votre salle après avoir téléchargé tous les emplois.
  • Le quatrième champs permet de spécifier votre nom d'utilisateur, ce champ n'as aucun rôle particulier.
  • Le dernier vous permet de changer le thème de votre application. Il existe deux thèmes. Sombre et Clair, ce dernier est configuré par défaut à l'installation.

Comment consulter un nouveau emploi

Une fois que Emploi Enastic est démarré, vous pouvez consulter les nouveaux emplois de temps en vous connectant au réseau local de l'Enastic.
Pour voir un emploi particulier, faites un clic droit pour voir la liste des classes, ensuite selectionner la salle dont vous voulez consulter l'emploi.

Comment imprimer un emploi

Pour imprimer un emploi de temps, clicquez tout simplement sur le bouton imprimer qui se trouve sur la barre d'outils. A ce moment une nouvelle fenêtre se présenteras a vous, choisissez l'emplacement ou vous voulez enregistrer et cliquer sur imprimer. Vous avez aussi la possibilité de modifier le nom de sortie de votre fichier.
